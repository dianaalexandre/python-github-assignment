Description
Reading Tracker
This program allows bookworms to track how many books they read every week. It allows individuals to keep track and eventually take count on how many books they will read per week and they can use this number for further calculation to meet a weekly goal.

Run Instructions
To use this code, insert the number of book(s) you have read. The program then calcuates how many books will be read this week. It is a simple tool to work towards building reading habits.
In order to run this code. Locate python_github_assignment in your terminal. Then use python3 app.py. 

Example Output:
How many books did you read today?: "3"
You are on track to read 21 books this week.
